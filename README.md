# :video_game: TabooGame

Simple party game where You need to describe words to Your partners without using forbidden ones.  


# :key:Technologies
Project is written in **Kotlin** with usage of:

* MVVM
* Kotlin Navigation
* Live Data
* Data Binding
* Shared Preferences
